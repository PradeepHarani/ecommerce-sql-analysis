# Raw Data

This project uses the Brazilian E-Commerce (Olist) dataset obtained from Kaggle.

The dataset consists of multiple related tables used for analysis:

- customers.csv → customer information and location details  
- orders.csv → order lifecycle and timestamps  
- order_items.csv → product-level transaction data  
- payments.csv → payment methods and transaction values  

The dataset is used as raw input and validated during SQL analysis.

Dataset source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
